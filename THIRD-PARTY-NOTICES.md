# Third-party notices

The **CSA Loom Console** and the CSA Loom platform are licensed under the **MIT
License** (see [`LICENSE`](LICENSE)). This file attributes the third-party
open-source packages bundled as **production dependencies** of the Console.

## How this file is produced and enforced

The authoritative, per-package attribution is **generated from the installed
production dependency tree** — it is not hand-maintained — by:

```bash
node scripts/ci/check-licenses.mjs --write THIRD-PARTY-NOTICES.md
```

This runs in CI on any change to the Console's `package.json` /
`pnpm-lock.yaml`, in **`.github/workflows/license-scan.yml`**, which:

1. Installs the Console production dependencies (`pnpm install --prod`).
2. Runs the **license allowlist gate** — the PR **fails** if any package's
   license falls outside the reviewed allowlist in
   `scripts/ci/check-licenses.mjs`. **GPL / AGPL and unknown/custom licenses are
   blocked** so a future viral-copyleft transitive dependency cannot silently
   contaminate the MIT distribution.
3. Regenerates the full per-package list below and uploads it as a build
   artifact (`third-party-notices`).

The Console image additionally ships a machine-readable attribution at
`/third-party-licenses.json` (generated in the Dockerfile from the same
production tree).

To refresh this file locally, run the command above from a checkout that has
`pnpm install`-ed `apps/fiab-console`, then commit the result.

## Allowed licenses (policy)

Permissive and weak/file-level copyleft licenses that are compatible with an MIT
distribution when the dependency is used as an unmodified library are allowed —
including **MIT, ISC, Apache-2.0, BSD-2/3-Clause, 0BSD, Zlib, Unlicense, CC0-1.0,
LGPL-3.0, LGPL-2.1, MPL-2.0, EPL-1.0/2.0, and CC-BY-4.0** (data/font assets). The
complete, authoritative list is the `ALLOWLIST` in
`scripts/ci/check-licenses.mjs`.

**Blocked:** GPL-\* and AGPL-\* (viral copyleft), and any `UNKNOWN` / `Custom`
license (these require a human review before shipping).

## Notable non-permissive components

The production tree currently includes these non-MIT (but allowed) licenses.
Each is used as an unmodified library and does not affect the MIT license of the
Console's own source:

| Component(s) | License | Notes |
| --- | --- | --- |
| `sharp` and its native/transitive deps | LGPL-3.0 (family) | Image processing; used as an unmodified shared library. |
| `elkjs` | EPL (Eclipse Public License) | Graph layout for canvas surfaces. |
| Several build/runtime utilities | MPL-2.0 | File-level copyleft; used unmodified. |
| Font / data assets | CC-BY-4.0 | Attribution-only content assets. |

The full, per-package list (grouped by license) is generated by the command
above and attached to the `license-scan` CI run. It is intentionally omitted
from this checked-in header when generated offline is not possible, so this file
never presents a stale or partial machine list as authoritative.
